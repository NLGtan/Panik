# Folder Purpose Map

## Root-level purpose

| Path | Purpose | Notes |
|---|---|---|
| `contracts/` | All Solidity contracts for Phase 0 logic. | Production + test mocks separated by subfolders. |
| `scripts/` | Deployment and utility scripts. | Used for Base Sepolia deployments and ABI export. |
| `deploy/` | Generated deployment outputs and deployment docs. | JSON outputs are consumed by frontend/backend config. |
| `test/` | Smart-contract behavior tests. | Includes happy + unhappy path checks. |
| `abi/` | Exported ABI artifacts for frontend integration. | Generated by `npm run export:abi`. |
| `architecture/` | Human-readable architecture documentation. | This folder. |

## `contracts/` detailed purpose

| Path | Purpose description |
|---|---|
| `contracts/PanikExecutor.sol` | Coordinates lock checking, sequencing, debt repayment, collateral withdraw, swap, and final USDC transfer. |
| `contracts/LockChecker.sol` | Reads Aave data provider and classifies assets as locked/unlocked for the current user context. |
| `contracts/libraries/SequenceLib.sol` | Builds ordered actions to enforce debt-first safety and deterministic processing order. |
| `contracts/adapters/AaveAdapter.sol` | Keeps Aave protocol calls isolated and reusable; validates post-repay health factor direction. |
| `contracts/adapters/SwapAdapter.sol` | Keeps UniversalRouter call shape isolated; enforces amount-out floor behavior. |
| `contracts/interfaces/*.sol` | Minimal interfaces for only what is required by the implementation. |
| `contracts/mocks/*.sol` | Test-only contract doubles for deterministic Hardhat tests. |

## `scripts/` detailed purpose

| Script | Purpose |
|---|---|
| `scripts/deploy.ts` | Deploy full Phase 0 stack and write `deploy/addresses.base-sepolia.json`. |
| `scripts/deploy-mock-oracle.ts` | Deploy only mock oracle for staged rollout. |
| `scripts/deploy-executor-only.ts` | Redeploy only `PanikExecutor` while reusing existing adapters/checker. |
| `scripts/export-abi.ts` | Export ABI-only files to `abi/` for frontend consumption. |

## `deploy/` file meaning

| File | Meaning |
|---|---|
| `deploy/addresses.base-sepolia.json` | Latest end-to-end deployment source of truth (addresses + config snapshot). |
| `deploy/panik-executor.base-sepolia.json` | Last executor-only redeploy output. |
| `deploy/mock-oracle.base-sepolia.json` | Last mock oracle deploy output. |
| `deploy/README.md` | Constructor arguments, deployment checklist, ABI export instructions. |

## `test/` purpose
- Verifies expected behavior under success and revert conditions.
- Confirms no forbidden admin surface exists.
- Confirms `SwapRouter02` is not referenced.
- Confirms rollback behavior for reverted flow.
